* Array Are Collection Of Elements of same data type.
* List Of Values 
* Each Value Has An Index.
* In Array Index Start From 0.


Some Of The Most Common Functions 

1- Push() : Add New Element At the End Of The Array.
2- Pop()  : Remove The Last Element Of the Array.
3- shift() : Remove the First Element in the array.
4- unShift() : Add New Element In the beginning of the Array.
5- slice() : Return The Subset Of the array
6- splice() : This Can Be used To Add Or Remove Element From An Array.
