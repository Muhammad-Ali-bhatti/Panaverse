In this Exercise We Will Learn About Inquirer.

Inquirer is Npm Package used to take input from the user.

in order to use Inquirer we should follow this steps.

npm init -y
tsc --init
npm i Inquirer
npm i @types/ Inquirer -D
change the target in tsconfig.json to latest 
change module to Next.js And Module Resolution To Next.js
Add Type = "Module" under Main Tag in Package.json file
